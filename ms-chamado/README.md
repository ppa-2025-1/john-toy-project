# Chamado Service

## Descrição
Serviço de gerenciamento de chamados desenvolvido por jaime.

## Funcionalidades
- Criação de chamados
- Consulta de chamados por ID
- Consulta de todos os chamados
- Alteração de situação de chamados
- Validações de negócio

## Tecnologias
- Spring Boot 3.x
- Spring Data JPA
- H2 Database
- REST API
