# User Service

## Descrição
Serviço de gerenciamento de usuários desenvolvido por Jaime Silva.

## Funcionalidades
- Criação de usuários
- Consulta de usuários
- Gerenciamento de perfis e roles
- Integração automática com serviço de tickets

## Tecnologias
- Spring Boot 3.x
- Spring Data JPA
- H2 Database
- REST API

- 8081
